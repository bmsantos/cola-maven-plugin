cola:compile

Full name:

com.github.bmsantos:cola-maven-plugin:0.1.0-SNAPSHOT:compile

Description:

(no description)

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: test.
  • Binds by default to the lifecycle phase: process-test-classes.

Optional Parameters

Name Type Since Description
excludes String[] - Exclude filters
ideBaseClass String - Cola Test Base Class. This is required in order to allow Cola Tests to run from IDEs such as Eclipse.
ideBaseClassTest String - Cola Test Base Class Test. The test name to be removed.
includes String[] - Inlude filters
log Boolean - Used to enable cola logging.
Default value is: true.
targetTestDirectory String - Used to enable cola logging.
User property is: project.build.testOutputDirectory.
